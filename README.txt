NextSkill — Deploy rápido
=================================

Estructura:
- index.html  -> Landing blanca
- game.html   -> Juego (tema azul)
- market_data.json -> Datos locales para el simulador

Cómo correr local:
1) Abre index.html con Live Server (o doble click).
2) El botón “Jugar ahora” abre game.html en una pestaña.

Cómo publicar (GitHub Pages):
1) Sube estos archivos a un repo público.
2) En Settings > Pages, Source: "Deploy from a branch" -> main /root.
3) Espera ~2 minutos, la URL pública mostrará index.html.
